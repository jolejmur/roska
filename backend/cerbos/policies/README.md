# Políticas de Cerbos

Este directorio contiene las políticas de autorización de Cerbos para el proyecto Roska Radiadores.

## Estructura

Las políticas se organizan por recursos:
- `resource_policies/`: Políticas basadas en recursos
- `principal_policies/`: Políticas basadas en principales (usuarios/roles)
- `derived_roles/`: Definiciones de roles derivados

## Ejemplo de uso

Ver `resource_policies/radiador_policy.yaml` para un ejemplo de política de recursos.
